📐 CATEGORIA 1: LAYOUT → Containers, Grid

Containers

A classe .container define uma largura máxima responsiva e centraliza o conteúdo com margens automáticas. Variações: .container-sm (≥576px) · .container-md (≥768px) · .container-lg (≥992px) · .container-xl (≥1200px) · .container-fluid (100% sempre)

.container — largura máxima centralizada

A classe .container-fluid faz o elemento ocupar 100% da largura disponível em qualquer tela.

.container-fluid — 100% da largura

A classe .container-sm ocupa 100% da largura em telas menores que 576px. Acima disso, tem largura fixa. Funciona igual para .container-md, .container-lg e .container-xl, cada um com seu breakpoint.

.container-sm — 100% abaixo de 576px, fixo acima

Sistema de Grades

A classe .row cria uma linha horizontal do grid. Sempre deve estar dentro de um container. A classe .col divide o espaço igualmente. Use .col-sm, .col-md, .col-lg para controle por breakpoint.

Use .col-{n} para definir o tamanho exato em colunas (de 1 a 12). A grade tem 12 colunas no total. Exemplos: .col-6 = 50% · .col-4 = 33% · .col-3 = 25% · .col-12 = 100%

Breakpoints: .navbar-expand-sm · .navbar-expand-md · .navbar-expand-lg · .navbar-expand-xl

.dropdown-toggle → abre submenu · .dropdown-menu → painel do submenu · .dropdown-item → item do submenu · .dropdown-divider → separador

📄 CATEGORIA 3: CONTEÚDO → Apresentação, Jumbotron, Imagens, Tabelas

Apresentação

Use .bg-{cor} para cor de fundo, .text-{cor} para cor do texto, .text-center para centralizar e .rounded para bordas arredondadas. Fundos: .bg-primary · .bg-success · .bg-warning · .bg-danger · .bg-info · .bg-dark · .bg-light
Texto: .text-white · .text-dark · .text-muted · .text-primary · .text-info
Bordas: .rounded · .rounded-lg · .rounded-pill · .rounded-circle · .rounded-0

Jumbotron

A classe .jumbotron cria uma caixa de destaque grande com fundo cinza claro, usada para chamar atenção ao conteúdo principal. Variação: .jumbotron-fluid → ocupa 100% da largura (sem bordas arredondadas)

Classe Jumbotron

Um jumbotron indica uma grande caixa cinza para chamar atenção extra para algum conteúdo ou informação especial. Dentro de um jumbotron você pode colocar praticamente qualquer HTML válido, incluindo outros elementos/classes do Bootstrap.

Outros textos.

Imagens

A classe .img-fluid torna a imagem responsiva: max-width: 100% e height: auto. A imagem encolhe em telas menores sem distorcer.

Use .d-block + .mx-auto para centralizar uma imagem. .d-block aplica display: block e .mx-auto define margens horizontais automáticas. Outros displays: .d-none · .d-inline · .d-inline-block · .d-flex  |  Responsivos: .d-sm-none · .d-md-block

A classe .rounded aplica bordas levemente arredondadas na imagem.

A classe .rounded-circle aplica border-radius: 50%, transformando a imagem em um círculo perfeito.

Use .float-left e .float-right para alinhar imagens. Adicione .clearfix no elemento pai para evitar colapso de altura. Responsivos: .float-sm-left · .float-md-right · .float-none

A classe .img-thumbnail adiciona borda fina, padding e bordas arredondadas. Ideal para galerias.

Tabelas

A classe .table aplica o estilo base do Bootstrap nas tabelas. Combine com modificadores para mais recursos. .table-striped → linhas alternadas (zebra)  ·  .table-bordered → bordas em todas as células
.table-hover → destaca linha ao passar o mouse  ·  .table-dark → fundo escuro  ·  .table-sm → células compactas
.thead-dark → cabeçalho escuro  ·  .thead-light → cabeçalho cinza claro

📝 CATEGORIA 5: FORMULÁRIOS → Empilhado, Inline, Input

Formulário Empilhado

A classe .form-group agrupa label e input com margin-bottom entre eles. A classe .form-control estiliza inputs, selects e textareas com borda, padding e largura 100%. Tamanhos: .form-control-lg · .form-control-sm
Checkbox/Radio: .form-check · .form-check-input · .form-check-label

Formulário Inline

A classe .form-inline coloca todos os campos em linha (horizontal) em telas ≥576px. Em telas menores, os campos empilham automaticamente.

Controle de Formulário: Input

Use .form-control em qualquer <input>, <select> ou <textarea> para aplicar o estilo Bootstrap padrão.

.animate__rubberBand → elástico  ·  .animate__shakeX → sacode horizontal  ·  .animate__swing → balança
Outros: .animate__zoomIn · .animate__slideInLeft · .animate__heartBeat · .animate__pulse · .animate__wobble
