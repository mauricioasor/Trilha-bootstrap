Classe jumbotron

Um jumbotron indica uma grande caixa cinza para chamar atenção extra para algum conteúdo ou informação especial. Dica: Dentro de um jumbotron você pode colocar praticamente qualquer HTML válido, incluindo outros elementos/classes do Bootstrap.

Alertas

Os alertas são criados com a classe .alert, seguida por classes de cores contextuais:

Sucesso! Esta caixa de alerta pode indicar uma ação bem-sucedida ou positiva.
Informação! Esta caixa de alerta pode indicar uma informação ou mudança neutra.
Atenção! Esta caixa de alerta pode indicar um aviso que precisa de atenção.
Perigo! Esta caixa de alerta pode indicar uma ação perigosa ou potencialmente negativa.
Principal! Indica uma ação importante.
Secundário! Indica uma ação um pouco menos importante.
Escuro! Alerta em cinza escuro.
Claro! Alerta em cinza claro.

Exemplo Básico de Navbar

Uma barra de navegação é um cabeçalho de navegação que fica localizado no topo da página.

A classe navbar-expand-xl|lg|md|sm determina quando a barra de navegação deve se reorganizar verticalmente (em telas extra grandes, grandes, médias ou pequenas).

Menus Suspensos (Dropdowns)

A classe .dropdown é usada para indicar um menu suspenso.

Use a classe .dropdown-menu para construir o menu suspenso.

Para abrir o menu suspenso, use um botão ou um link com a classe .dropdown-toggle e o atributo data-toggle="dropdown".
